Файл нужно открыть до инструкции try,
потому что если инструкция open сама по себе вызовет ошибку,
то файл не будет открываться для последующего закрытия.

Еще один подход — использовать инструкцию with, которая упрощает
обработку исключений с помощью инкапсуляции начальных операций,
а также задач по закрытию и очистке.
